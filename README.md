# Todo List React App 📝

A simple and responsive Todo List application built with React ,tailwind , React icons and Vite. Add, delete, and manage your daily tasks with ease.

---

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Built with React + Vite
